# 🌸 NityaVerse 🌸

Welcome to **NityaVerse**—a beautifully crafted, real-time study planner designed with love, built as a hobby project to help my sister conquer her competitive exams. 

When preparing for major milestones, the journey can feel overwhelming. I wanted to create a calm, aesthetic, and completely stress-free digital environment where she could organize her thoughts, track her progress, and feel motivated every single day without the clutter of traditional productivity tools.

## ✨ Why NityaVerse?
The application is wrapped in a soft, minimalistic aesthetic inspired by pastel themes, providing a serene workspace. But beneath the soft UI is a powerful, real-time engine:
- **Zero-Friction Tracking**: Break down massive subjects into digestible chapters, daily missions, and weekly overviews.
- **Embedded Motivation**: Visual progress bars, streak trackers, and gentle reminders (*"You are closer than yesterday ✨"*) keep morale high.
- **Real-Time Co-Op Connection**: Powered by a custom-built cloud-sync proxy. I can watch her progress update live on my screen from miles away, cheering her on as she checks off her goals—without a single page refresh.
- **All-in-One Space**: Attach multiple video lecture links directly alongside practice trackers and 3-stage revision loops.

## 🚀 Features
- **Personalized Dashboards**: Subject-by-subject completion metrics and streak tracking.
- **Chapter Deep-Dives**: Detailed grids to monitor watch-status, practice completion, and revision cycles.
- **Multi-Media Support**: Scalable video link attachments per topic.
- **Daily & Weekly Previews**: Dedicated spaces to prioritize the top tasks of the day and preview the week ahead.
- **Live Syncing Engine**: Secure cloud proxying using Vercel Serverless Functions to instantly mirror state across devices via a single co-op link.

## 🛠️ Built With
- **Frontend**: React.js & Vite for lightning-fast capability.
- **Styling**: 100% Custom Vanilla CSS Design System built to recreate a serene, pastel aesthetic.
- **Backend / Sync**: Vercel Serverless API (`/api/jsonblob.js`) routing to public JSON cloud spaces.
- **Icons**: Lucide-React.

## 🌟 The Goal
This isn't just a planner; it's a digital hug and a cheerleading squad built out of code. It's proof that technology can be deeply personal, aesthetic, and incredibly functional all at once.
